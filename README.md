# 🎮 VOIDLACE: OVERDRIVE

Киберпанк 3D шутер-аркада на JavaScript с использованием Three.js.

## 🕹️ Геймплей

Летите через цифровой туннель, уничтожайте врагов и набирайте очки! 

**Механика:**
- **BOOST (W / BOOST Button)** - ускоряйтесь вперед
- **FIRE (SPACE / FIRE Button)** - стреляйте в врагов
- Каждый уничтоженный враг = +100 очков
- Каждое столкновение = -20 HP
- Game Over при HP = 0

## 🎨 Визуальный стиль

- Киберсетка-туннель
- Светящийся игрок (октаэдр)
- Враги (вращающиеся красные кубы)
- Эффекты частиц при взрывах
- Неоновые цвета (#00ffff, #ff00ff, #ff0055)
- Монохромный шрифт Courier New

## 🚀 Технологии

- **Three.js** - 3D графика
- **HTML5 Canvas** - рендеринг
- **Vanilla JavaScript** - логика игры
- **Mobile Touch Controls** - поддержка мобилки

## 📱 Управление

### На ПК:
- **W** - ускорение
- **SPACE** - стрельба

### На мобильном:
- **BOOST** - кнопка ускорения
- **FIRE** - кнопка стрельбы

## 🎮 Как играть

1. Откройте `index.html` в браузере
2. Используйте управление (W + SPACE на ПК или кнопки на мобильном)
3. Уничтожайте врагов и набирайте очки!
